# 🦀 Clawd Desktop

귀여운 픽셀 크랩 데스크탑 펫 — Windows 전용

---

## ⚡ .exe 받는 방법 (아무것도 설치 불필요)

### 1단계 — GitHub 저장소 만들기

1. https://github.com 로그인
2. 우상단 **+** → **New repository**
3. 이름: `clawd-desktop` / Public 선택 / **Create repository**

### 2단계 — 파일 업로드

저장소 페이지에서 **"uploading an existing file"** 클릭  
→ 이 zip 안의 **모든 파일과 폴더**를 드래그앤드롭  
→ **Commit changes** 클릭

> ⚠️ `.github/workflows/build.yml` 파일이 반드시 포함되어야 합니다!

### 3단계 — 자동 빌드 기다리기

업로드하면 **GitHub Actions**가 자동으로 실행됩니다.  
저장소 → **Actions 탭** → 초록 체크 뜰 때까지 대기 (약 5~10분)

### 4단계 — .exe 다운로드

저장소 → **Releases** → `Clawd-Setup-1.0.0.exe` 다운로드  
→ **더블클릭으로 설치 & 실행!** 🎉
